Cómo escribimos CSS

Hay varias maneras:

  • Selector de etiqueta (es genérico)
  • Selector de clase
  • Selector de id
  • Selector de combinados
  • Selector de hijos
  • Selector de descendientes
  • Selector de hermanos
  • Selector de hermanos adyacentes

Selectores:

Selector de etiqueta (es genérico)

Cogemos el nombre de la etiqueta global y ponemos un estilo. Afectará a todo aquello que esté en esa etiqueta

                body {
                    color: blue;
                    family-font: italic; 
                }

Selector de clase

Le asigamos a una etiqueta una clase para darle un estilo concreto. 

                    
En HTML

                        <figure class="centrado">
                            <img> [...]
                        </figure>
                    
En CSS

                        .centrado {
                            text-aling: center;
                        }

Selector de id (pero no se usa por buenas prácticas)

Le asigamos a una etiqueta una clase para darle un estilo concreto.

                    
En HTML

                        <figure id="centrado">
                            <img> [...]
                        </figure>
                    
En CSS

                        #centrado {
                            text-aling: center;
                        }

Selector de descendientes

Utiliza el símbolo del espacio Des esta manera indicamos cualquier objeto que sea descendiente de un nivel. Cogemos el descendiente x de x padre. Cogerá todos aquellos descendientes estén dentro de otro bloque o no.

Ejemplo: 

                
En HTML

                    <div> 
                        <p> Lorem ipsum dolor, sit amet consectetur adipisicing elit. Laudantium, accusantium eum q
                    </div>
                
En css

                    div p {
                        background-color: red;

Selector de hermano

Aparece en CC3. Se representa con ~. Con esto queremos decir que cogerá las cosas que estén al mismo nivel que el div, por lo que afectará a los párrafos siguientes a div en éste caso

                    <p>

                    <p>
                    Lorem ipsum dolor sit amet consectetur, adipisicing elit. Perferendis mollitia molestiae dolore aspernatur temporibus commodi veritatis, sed

Selector de hermano adyacente

Se representa con + . Adyacente es justo el que va a continuación de la etiqueta div, en el caso anterior, sólo afectará al primer párrafo. Siempre habrá un adayacente.

                    
En HTML

                        .combinado div+p {

Selector de atributo

Selector de atributos: title. Utiliza los dos corchetes para indicar que lo que viene es un atributo []. De esta manera todos los elementos que tengan el atributo title, tendrán un background-color rojo. 

            
En HTML

            <p title="Parrafo 1">Lorem ipsum dolor sit amet consectetur adipisicing elit. Quisquam n.</p>
            <p title="Parrafo 2">Lorem ipsum dolor sit amet consectetur adipisicing elit. </p>
            
En CSS

            [title] {
                background-color: red;
            }

^ Con este simbolo indicamos que algo que empiece por x saldrá todo aquello que empiece por eso.

Ejemplo:

                [title^="Parrafo"] {
                    background-color: red;
                }

De esta manera saldrán nuestros dos párrafos.

* "Atributo que contiene ...

Ejemplo:

                [title*="rraf"] {
                    background-color: red;
                }

De igual manera nos saldrán nuestros dos párrafos.

$ "Termina por..."

p>Ejemplo:

                [title$="1"] {
                    background-color: red;
                }

En este caso sólamente se verá afectado el párrafo número 1.

!important, hace que una propiedad tenga más importancia que cualquier especifidad. Se coloca entre el valor y el ;.

Pseudo elementos

Los pseudo elementos sirve para dar orden a elementos, gracias a dos clases: 

            .pseudo-elem p:nth-child(2){
                text-decoration: underline;
            }

En () debes especificar la posición que ocupa entre "los hijos" de section en este caso. Todos los párrafos que sena segundo hijo de alguien tiene que seguir esta norma, por eso el último párrafo aparece subrayado, porque es el segundo hijo de div

En el caso de:

            .pseudo-elem p:nth-of-type(2){ 
                font-style: italic;
            }

Cuenta párrafos y coge el segundo párrafo independientemente de si hay mas tipos de tiquetas, las salta. En cambio en el anterior, cuenta todo y si es párrafo y segundo hijo lo pinta.

            .pseudo-elem li:nth-of-type(2n){ /* da igual poner cualquiera de las anteriores (nth-of-type ó nth-child) */
                background-color: rgb(202, 236, 152);
            }

n se va multiplicando por 2 la posición por dos. Otra forma es poner "Even" en 2n

Para los impares:

            .pseudo-elem li:nth-of-type(2n+1){ 
                background-color: rgb(233, 152, 236);
            }

n se va multiplicando por 2 la posición por dos más 1. Otra forma es poner "odd" en 2n
