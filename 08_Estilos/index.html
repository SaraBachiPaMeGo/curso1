Cómo escribimos CSS

Hay varias maneras:

  • Selector de etiqueta (es genérico)
  • Selector de clase
  • Selector de id
  • Selector de combinados
  • Selector de hijos
  • Selector de descendientes
  • Selector de hermanos
  • Selector de hermanos adyacentes

Selectores:

Selector de etiqueta (es genérico)

Cogemos el nombre de la etiqueta global y ponemos un estilo. Afectará a todo aquello que esté en esa etiqueta

                body {
                    color: blue;
                    family-font: italic; 
                }

Selector de clase

Le asigamos a una etiqueta una clase para darle un estilo concreto. 

                    
En HTML

                        <figure class="centrado">
                            <img> [...]
                        </figure>
                    
En CSS

                        .centrado {
                            text-aling: center;
                        }

Selector de id (pero no se usa por buenas prácticas)

Le asigamos a una etiqueta una clase para darle un estilo concreto.

                    
En HTML

                        <figure id="centrado">
                            <img> [...]
                        </figure>
                    
En CSS

                        #centrado {
                            text-aling: center;
                        }

Selector de hijos

Su símbolo es el ">". Aquellos que sólo sean hijos directos de la clase pero no sus descendientes. Se usa cuando quiero restringir algo y que afecte a los elementos inmediatos sin tocar su descendencia. También se utilizan mucho en los menús y submenús.

                
En HTML


                
En CSS

                .combinadores>p {
                    color:black;
                }

Selector de descendientes

Utiliza el símbolo del espacio Des esta manera indicamos cualquier objeto que sea descendiente de un nivel. Cogemos el descendiente x de x padre. Cogerá todos aquellos descendientes estén dentro de otro bloque o no.

Ejemplo: 

                
En HTML

                    <div> 
                        <p> Lorem ipsum dolor, sit amet consectetur adipisicing elit. Laudantium, accusantium eum quasi officia ut quaerat pariatur perferendis numquam voluptates labore nesciunt recusandae blanditiis totam consequatur itaque quisquam soluta tenetur? Consequatur!
                    </div>
                
En css

                    div p {
                        background-color: red; 
                    }

Selector de atributo

Selector de atributos: title. Utiliza los dos corchetes para indicar que lo que viene es un atributo []. De esta manera todos los elementos que tengan el atributo title, tendrán un background-color rojo. 

            
En HTML

            <p title="Parrafo 1">Lorem ipsum dolor sit amet consectetur adipisicing elit. Quisquam nisi accusantium repellendus ipsum quia molestias placeat. Laboriosam, adipisci. Sapiente, animi nisi praesentium culpa officiis sint fugit vitae omnis accusantium labore.</p>
            <p title="Parrafo 2">Lorem ipsum dolor sit amet consectetur adipisicing elit. Libero maxime dicta saepe similique excepturi quasi unde temporibus nihil at. Nemo enim, illo sed ipsum tempora tenetur. Ex cupiditate obcaecati recusandae.</p>
            
En CSS

            [title] {
                background-color: red;
            }

^ Con este simbolo indicamos que algo que empiece por x saldrá todo aquello que empiece por eso.

Ejemplo:

                [title^="Parrafo"] {
                    background-color: red;
                }

De esta manera saldrán nuestros dos párrafos.

* "Atributo que contiene ...

Ejemplo:

                [title*="rraf"] {
                    background-color: red;
                }

De igual manera nos saldrán nuestros dos párrafos.

$ "Termina por..."

p>Ejemplo:

                [title$="1"] {
                    background-color: red;
                }

En este caso sólamente se verá afectado el párrafo número 1.
